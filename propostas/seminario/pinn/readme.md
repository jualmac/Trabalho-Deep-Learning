# Artigos:
* Automatic network structure discovery of physics informed neural networks via knowledge
  distillation(https://doi.org/10.1038/s41467-025-64624-3)
* * Publicado na Nature Communications;
* * Código Aberto: https://github.com/ZitiLiu/Psi-NN -> Importante para as nossas futuras tentativas de reprodução;
* * O Ψ-NN propõe descobrir automaticamente estruturas de PINNs usando destilação de conhecimento: uma rede professora aprende a física do problema e uma rede estudante aprende suas saídas e gradientes;
* * Depois, os pesos da estudante são analisados por agrupamento para reconstruir uma arquitetura mais compacta, eficiente e interpretável;
* * Apesar de dialogar com descoberta automática de arquiteturas, não é Neuroevolução, pois não usa população, seleção, mutação ou crossover. É um treinamento por gradiente e análise matemática da rede treinada;
