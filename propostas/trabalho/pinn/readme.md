# Trabalho
* Para o trabalho, minha ideia principal seria explorar arquiteturas de Physics-informed Neural Networks (PINN);

* PINNs são redes neurais que aprendem soluções para problemas físicos incorporando as equações do sistema diretamente no processo de treinamento. Em vez de depender só de dados observados, elas também são penalizadas quando suas previsões violam leis físicas, como equações diferenciais, condições iniciais ou condições de contorno.
